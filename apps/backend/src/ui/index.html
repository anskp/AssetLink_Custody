Integration Guide

1. Base URL Configuration

For all local integrations, use the following base URL:

http://localhost:3000/v1

2. Required Authentication Headers

Header Description
x-api-key Your Public Infrastructure Key
x-signature HMAC-SHA256 signature of the request
x-timestamp Current Unix timestamp (seconds)
x-user-id End-user identifier (Required for mutations)

3. Signature Construction

The signature is an HMAC-SHA256 hash created using your Secret Key.

Payload template: METHOD + PATH + TIMESTAMP + BODY_STRING

Example Path: /v1/custody/link (Must include /v1 prefix)

4. Local Express.js Integration (Port 5000)

If your project is running on port 5000, it can communicate directly with AssetLink on port 3000. No proxy or server-side CORS setup is needed for backend-to-backend calls.
